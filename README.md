
Este repositório contém a implementação de uma Árvore Binária de Busca (BST) desenvolvida em Java. O projeto foi criado para consolidar conceitos fundamentais de estruturas de dados, manipulação de nós na memória e algoritmos de busca recursiva.

---

## Execução do Projeto

Para rodar o sistema na sua máquina local, certifique-se de ter o Java JDK instalado. Em seguida, abra o terminal no diretório raiz do projeto e execute os seguintes comandos:

```bash
# Compila os arquivos Java
javac *.java

# Executa o programa principal
java Main
